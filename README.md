## 🧠 Brain Tumor Detection using Deep Learning

## 📌 Overview

This project is a web-based Brain Tumor Detection system developed using **Flask** and **TensorFlow/Keras**. The application allows users to upload an MRI brain scan image and predicts whether the image contains a brain tumor.

The trained deep learning model classifies the MRI image into one of the following categories:

* Glioma
* Meningioma
* Pituitary Tumor
* No Tumor

The application also displays the prediction confidence along with the detection status.
